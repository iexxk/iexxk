运行buane将YAML文件转换为JSON Ignition配置。
点击`Actions->执行butane转换->run workflow`会将fedora-core-config.yaml文件转成fedora-core-config.ign
然后获得ign的链接https://raw.githubusercontent.com/iexxk/iexxk/refs/heads/main/fedora-core-config.ign，就可以通过该链接的配置进行安装fedora-coreos的系统了。
